📁 GitHub & Google Drive

💡 Este sistema é um arquivo HTML único. Salve-o no GitHub Pages para acesso online e use o botão abaixo para exportar para o Google Drive.
1. Crie um repositório no GitHub chamado "folha-pagamento" 2. Faça upload deste arquivo como "index.html" 3. Vá em Settings > Pages > Source: main branch 4. Acesse: https://seu-usuario.github.io/folha-pagamento 5. Os dados ficam no localStorage do navegador de cada usuário 6. Use "Exportar Backup" para salvar e compartilhar dados
